Principais TAGs HTML

Headings

Importante para definir ordem de relevância dos títulos e auxiliar buscadores para rankiamento de página.

  • h1 - Titulo de Primeira Ordem
  • h2 - Titulo de Segunda Ordem
  • h3 - Titulo de Terceira Ordem
  • h4 - Titulo de Quarta Ordem
  • h5 - Titulo de Quinta Ordem
  • h6 - Titulo de Sexta Ordem

    Conteúdo textual

    1. tag 'div' - Container genérico para "restrição" de conteúdo. Deve ser utilizado somente quando não houver outro elemento de semântica, como 'nav' ou 'article';
    2. tag 'p' - Inicialização de parágrafo;
    3. tag 'ul' - Para criação de lista desordenada. O Icone que precede a lista é manipulável pelo CSS;
    4. tag 'ol' - Para criação de lista ordenada. O formato de ordenação (numeral, romano, etc.) que precede a lista é manipulável pelo CSS;
    5. tag 'menu' - Tag semântica para criaçãod e uma lista desordenada. O formato de ordenação (numeral, romano, etc.) que precede a lista é manipulável pelo CSS;
    6. tag 'li' - É utilizado para respresentar um item que faz parte de uma lista, seja ela ordenada ou não;
