    Conteúdo textual

    1. tag 'div' é uma tag de contexto genérico em bloco - Container genérico para "restrição" de conteúdo. Deve ser utilizado somente quando não houver outro elemento de semântica, como 'nav' ou 'article';
    2. tag 'span' é uma tag de contexto genérico in line - Container genérico para "restrição" de conteúdo. Deve ser utilizado para atribuir e manipular trachos inline;
    3. tag 'p' - Inicialização de parágrafo;
    4. tag 'ul' - Para criação de lista desordenada. O Icone que precede a lista é manipulável pelo CSS;
    5. tag 'ol' - Para criação de lista ordenada. O formato de ordenação (numeral, romano, etc.) que precede a lista é manipulável pelo CSS;
    6. tag 'menu' - Tag semântica para criaçãod e uma lista desordenada. O formato de ordenação (numeral, romano, etc.) que precede a lista é manipulável pelo CSS;
    7. tag 'li' - É utilizado para respresentar um item que faz parte de uma lista, seja ela ordenada ou não;
    8. tag 'br' de broke - É utilizado para realizar uma quebra de linha;
    9. tag 'hr' - É utilizado para realizar uma quebra temática, ou seja, separa por meio de uma linha textos dentro de uma página html;

    Semânticas Textuais Inline

    1. tag 'strong' - Substitui a tag 'b' de bold. A ideia é marcar como relevante a um trechoque deve ser destacado;
    2. tag 'em' de emphasis - Substitui a tag 'i' de italic. A ideia é dar ênfase a um trecho que deve ser destacado;
    3. tag 'a'- Para criação de um LINK. A tag 'a' contém atributos como o 'href' no qual vc atribui uma url que conduzirá a página de interesse ao clicar no link. Já o atributo 'target' permite entre outras, definir se a página aberta pelo link será renderizada na mesma aba de navegação (self) ou em outra aba (blank);
    4. tag 'img' de image - Serve para inserção de imagens em um texto html. Mas essa tag será abordada com mais exclusividade posteriormente;
    5. tag 'del' e 'ins' de delete e insert - Serve para registro de alteração de parte de um texto, ficando assim, e inserção de uma nova parte, ficando assim, respectivamente;
    6. tag 'q' de 'quote' ou citação - Serve para fazer uma citação. Lembrando que o atributo 'cite' pode conter uma url ou mesmo um arquivo;
    7. tag 'code' para destaque de códigos como em linguagens de programação por exemplo. Veja a aplicação python a seguir: print('Hello, World!')
    8. tag 'pre' de pre format. Fundamental para manutenção de um texto conforme formatação de interesse. Se por exemplo eu quiser demonstrar um codigo dentro da tag 'code', caso haja espaços, quebras de linha, etc., minha tag 'code tem que vir dentro de uma tag 'pre', veja:

    9. tag 'sup' - Para operações matemáticas manipulações de textos e simbologias, exemplo: ax2 + bx + c;
    10. tag 'sub' - Para operações matemáticas, manipulações de textos e simbologias exemplo: H2O;
