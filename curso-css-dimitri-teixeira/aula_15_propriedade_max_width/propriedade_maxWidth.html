Propriedades CSS Max Width

Propriedade Width

Paragrafo com 800 px de largura. Repare que ao diminuir a tela do navegador com menos de 800px aparce uma barra de rolagem horizontal.

Propriedade Width: Auto.

Neste caso o bloco se adapta ao tamanho da tela. Perceba que, neste caso, sem barra de rolgem.

Propriedade Max-Width

Paragrafo com no máximo 800 px. Neste caso, a propriedade Max-Width faz o bloco se adapatar ao tamanho da tela. Assim, o bloco terá no máximo 800px de largura ou a quantidade que o dispositivo reduzido tiver de tamanho de display.
