Tipos de Display

Display Block

Comportamento padrão das tags HTML:

  • div
  • h1 ao h6
  • p
  • form
  • header
  • section

Características:

  • Por padrão ocupam toda a largura da página apresentando um comportamento de bloco.
  • Por isso, sempre que for utilizada uma tag HTML tipo bloco seguida da outra, cada uma será alocada em uma linha distinta.
  • Atributos como width, height e margin afetam essas tags.

Display Inline

Comportamento padrão das tags HTML:

  • a
  • span
  • img

Características:

  • Por padrão ocupam somente a largura do seu conteúdo.
  • Por isso, sempre que for utilizada uma tag HTML tipo inline seguida da outra, cada uma será alocada na mesma linha, uma ao lado da outra.
  • Atributos como width, height e margin não afetam essas tags. Já o padding afeta pois altera o comportamento do preenchimento na propria linha.

Exemplo de comportamento:

Link 01 Link 02 Link 03

Display Inline-Block

Comportamento padrão das tags HTML que receberem a propriedade CSS "display:inline-block"

Características:

  • Deste modo, as tags HTML herdam as características dos displays block e inline.
  • Por isso, sempre que for utilizada uma tag HTML tipo inline-block seguida da outra, cada uma poderá será alocada na mesma linha, mas também poderá ter atribuições como especificação do width, height, margin, etc.
  • Para iso o display deverá ser declarado como inline-block.
