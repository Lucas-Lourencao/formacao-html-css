Bloco com Position Absolute - Container sem position declarado

O elemento HMTL "filho" assume uma posição tendo como referencia um elemento HTML "pai". Contudo, para isso o elemento "filho" deve estar contido dentro de um elemento "pai" que contenha um position declarado e diferente do static.

Passe o mouse sobre o bloco abaixo.

Perceba que neste caso, o bloco vermelho foi posicionado no início da página, pois tomou o elemento body como referência, devido a ausência de position em um elemento pai mais próximo.

Caso o position do elemento "pai" não seja declarado, o elemento "filho" se comportará tendo como referência um elemento "pai" mais externo que contenha o position declarado, ou então utilizará o body como referência.
