O Z-Index permite manipular os elementos HTML de modo que seja possível controlar a sobreposição entre eles.

Quanto maior o Z-Index, mais a frente estará o elemento.

O valor de Z- Index pode assumir um valor negativo. Perceba que ao criar uma div 4, atribuimos o z-index = -1. Deste modo, o bloco foi posicionado atrás do background do elemento pai, que até então era o elemento mais ao fundo do bloco. Você conseguirá ver o bloco roxo se passar o mouse sobre o bloco azul.
