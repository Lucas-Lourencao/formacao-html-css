Propriedade Width e Height

Height - Altura de um elemento:

Este parágrafo terá uma altura de 100px

Width - Largura de um elemento:

Este parágrafo terá uma largura de 300px

Width - Definindo uma largura máxima de um elemento - não responsivo:

Este parágrafo terá uma largura fixa de 800px. Repare que ao reduzir o display da pagina aparece uma barra de rolagem horizontal. Além disso, o texto do parágrafo não se adapta a tela, sendo necessário rolar a barra de rolagem para conseguir realizar sua leitura.

Max Width - Definindo uma largura máxima de um elemento - responsivo:

Este parágrafo terá no máximo uma largura de 800px. Diferentemente da definição do width de 800px, a propriedade max-width em 800px ajusta a página quando a mesma é reduzida de modo que não aparece uma barra de rolagem na parte horizontal da página. Para verificar esssa questão, basta comentar o parágrafo anterior.

Max Width - Auto:

Devido ao atributo Auto do Max-Width, este parágrafo sempre terá o tamanho do display em que ele for executado.
